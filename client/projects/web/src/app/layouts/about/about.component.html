About This App

This is the GolfPicks application.
Technologies Used:
  • The web app is implemented using Angular.js
  • The server side is implemented using Node.js
  • DynamoDB is used as the backend database.
  • Weather forecast data supplied by the OpenWeather API
  • Maps data supplied by Google
  • Hosted on AWS
