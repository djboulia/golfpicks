GolfPicks Mobile Web

This is the GolfPicks application built on the IBM Cloud.

Technologies Used

  • The web app is implemented using Angular.js with the mobile app based on the Ionic framework.
  • The server side is implemented using StrongLoop with Node.js
  • Cloudant is used as the backend NOSQL database.
  • Weather forecast data supplied by the Weather Company API
